# 🎮 Pacman en Java

Un clone de Pacman développé en Java avec la bibliothèque Swing.

## Fonctionnalités
- Déplacement du Pacman avec les touches directionnelles
- Fantôme (Binky) avec IA
- Système de score
- Plateau de jeu personnalisé

## Technologies
- Java
- Java Swing (interface graphique)

## Comment lancer le jeu
```bash
javac *.java
java main
